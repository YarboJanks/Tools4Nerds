# Tools 4 Nerds

An ESO addon by **@YarboJanks** that bundles several quality-of-life combat indicators into one lightweight package.

**Requires:** [LibAddonMenu-2.0](https://www.esoui.com/downloads/info7-LibAddonMenu.html)

---

## Features

### CC Immunity Tracker
Displays a configurable on-screen label when your character is immune to crowd control effects, so you always know when CC will and won't land.

- Toggle on/off
- Customizable font size and color

### Block Indicator
Shows an on-screen label while you are actively blocking.

- Toggle on/off
- Customizable color

### Crit Hit Marker
Displays a visual marker on screen when you land a critical hit.

- Toggle on/off
- Customizable size and color

### Auto Queue Accept
Automatically accepts dungeon and activity queue pop-ups so you never miss a queue.

- Toggle on/off

### Global Cooldown (GCD) Overlay
Adds a cooldown animation directly over each action bar slot during the global cooldown, giving you a clear visual indicator of when your next ability is available. Fully replaces the standalone "Show Global Cooldown" addon.

- **Animation style** — choose from three styles:
  - Ascending (bottom to top)
  - Descending (top to bottom)
  - Radial
- **Icon desaturation** — grey out ability icons during the GCD
- **Ready animation** — flash the slot when the GCD expires
- **Potion cooldown** — optionally show the GCD overlay on consumable slots (off by default)

---

## Settings

All options are configurable in-game via:

**ESC → Settings → Addons → Tools 4 Nerds**

---

## Installation

1. Download and extract to `Elder Scrolls Online\live\AddOns\Tools4Nerds`
2. Make sure [LibAddonMenu-2.0](https://www.esoui.com/downloads/info7-LibAddonMenu.html) is also installed
3. Enable the addon in the ESO addon manager
